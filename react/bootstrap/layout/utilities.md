# Utilities for layout

Dla szybszego tworzenia responsywnych i przyjaznych dla urządzeń mobilnych stron, Bootstrap zawiera dziesiątki klas pomocniczych do pokazywania, ukrywania, wyrównywania i rozmieszczania treści.

## Changing `display`

Użyj naszych `display utilities` aby responsywnie przełączać popularne wartości właściwoście `display` Możesz je łączyć z systemem siatki, treścią lub komponentami, aby pokazywać lub ukrywać je na określonych widokach.

## Flexbox options

Jeśli potrzebujesz dodać `display: flex` do elementu, użyj klasy `.d-flex` lub jednej z jej responsywnych wersji (np., `.d-sm-flex`). Ta klasa lub wartość `display` jest potrzebna, aby korzystać z dodatkowych [flexbox utilities](/docs/5.3/utilities/flex) do ustawiania rozmiarów, wyrównania, odstępów i innych właściwości. 

## Margin and padding

Użyj `margin` i `padding` `spacing utilities` to control how elements and components are spaced and sized. Bootstrap includes a six-level scale for spacing utilities, based on aaby kontrolować rozmieszczenie i rozmiar elementów oraz komponentów. Bootstrap zawiera sześciopoziomową skalę dla narzędzi spacing, opartą na wartości domyślnej  `$spacer` równej `1rem`. Wybieraj wartości dla wszystkich widoków (np. `.me-3` dla `margin-right: 1rem` w układzie LTR) lub użyj responsywnych wariantów, aby targetować konkretne widoki (np., `.me-md-3` dla`margin-right: 1rem` — w LTR — zaczynając od breakpoint `md` ).

## Toggle `visibility`

eśli nie jest potrzebne przełączanie `display` isn’t needed, ożesz zmieniać `visibility` elementu za pomocą naszych `visibility utilities` Niewidoczne elementy nadal wpływają na układ strony, ale są wizualnie ukryte przed odwiedzającymi.
